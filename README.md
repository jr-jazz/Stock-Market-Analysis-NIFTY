NIFTY Indices Performance Analysis & Risk Insights

A comprehensive data analytics project analyzing historical performance, trends, volatility, and inter-relationships of three major Indian equity indices: NIFTY50, NIFTY100, and NIFTY500.

Built as part of a Big Data Analytics academic assignment — includes database design, SQL aggregation, Python-based data mining, statistical modeling, and executive-level visualisations.

Project Overview

Objective
Understand long-term growth trends, risk (volatility) profiles, co-movement patterns, and diversification benefits across large-cap (NIFTY50), large+mid-cap (NIFTY100), and broad-market (NIFTY500) indices to support better portfolio decisions and client advisory in the Indian equity market.

Key Techniques Used
- Relational database design & normalisation (MySQL)
- SQL data aggregation (daily OHLC from 5-min data)
- Linear regression (simple & multiple) for trend and volatility forecasting
- Rolling 30-day volatility analysis
- Pearson correlation & co-movement study
- Distribution analysis (box plots)
- Financial candlestick visualisation
- Interactive prediction with user input

Main Findings
- NIFTY500 shows stronger long-term growth (~11.7% annualised) but significantly higher volatility
- Very high correlation (0.94–0.98) between indices — NIFTY100 serves as a strong, lower-risk proxy for NIFTY500
- Volatility spikes disproportionately affect broader indices (especially during crises)
- Clear evidence of diversification premium in mid/small-cap exposure

Technologies & Tools
- Database: MySQL 8.0 (relational, normalised schema)
- Data Processing & Analysis: Python 3.11, pandas, numpy, scikit-learn
- Visualisation: seaborn, matplotlib, mplfinance
- Other: Power BI (supplementary visuals)
